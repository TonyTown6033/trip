## 路线风险结论

1. 主路线顺序暂不调整：`昆明 -> 大理 -> 沙溪/剑川 -> 丽江 -> 泸沽湖 -> 香格里拉/德钦 -> 腾冲/芒市 -> 普洱/景迈山/西双版纳 -> 建水/元阳/蒙自/普者黑 -> 昆明` 仍符合交通连续性和高海拔逐步适应。
2. 需要调整的是执行方式，不是立即倒排路线：雨季风险从 `6月` 开始显著增加，尤其是德钦/梅里、腾冲/芒市、普洱/景迈山/西双版纳、元阳/普者黑等山路或湖区点。
3. 泸沽湖、德钦/梅里、腾冲/芒市是第一批可删减段：连续降雨、地质灾害预警、道路管制、预算压力任一出现，就缩短或跳过。
4. 虎跳峡、雨崩、长距离山路、夜间班车不作为刚性目标；只在连续 2-3 天天气稳定、无预警、交通正常时考虑。
5. 建水、蒙自、昆明可作为雨天缓冲城市；如果后段南部/西部连续强降雨，可提前转入红河州城市段，再回昆明。
