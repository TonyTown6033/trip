# Bookings

记录人工确认后的预订信息。

不要保存敏感信息：

- 身份证号
- 护照号
- 银行卡号
- 完整手机号
- 支付密码

建议记录：

- 订单平台
- 订单类型
- 日期
- 城市
- 名称
- 金额
- 取消/退改规则
- 确认状态
