# Attractions

记录各城市景点/POI 候选信息。

## 数据来源

- 高德地图 API（POI 搜索）
- 小红书口碑
- 人工补充

## 建议记录

- 城市
- 景点名称
- 类型（自然/人文/博物馆/市集/徒步）
- 门票价格
- 开放时间
- 预约要求
- 推荐指数
- 备注（避坑/时令/交通）
