# Buy List

装备采购按优先级执行。预算紧张时，只买 `P0`，`P1` 看已有装备情况，`P2` 尽量不买。

## 购买参考

- 背包：优先看 `迪卡侬 MT100 50L` / `MT100 EASYFIT 50L` 或同级 `廉航包`。`2026-05-12` 与理塘王佶沟通结论：新包价位已低，二手没必要；先买一个基础款，后续长线高强度使用再升级。
- 已购：`迪卡侬 MT100 EASYFIT 50L`，实付 `469 CNY`；毛巾实付 `19.9 CNY`。
- 防雨罩：基础 `MT100 50L` 官方说明为不含防雨罩，需要另买；`MT100 EASYFIT 50L` 官方说明为含防雨罩。单独购买的防雨罩已退，暂不另买。
- 规格：目标 `40-50L`，优先舒适背负；不要把 `MT100 50L` 当作可靠廉航登机包。官方页面显示 `MT100 EASYFIT 50L` 使用尺寸约 `58 x 26 x 26 cm`，基础 `MT100 50L` 约 `64 x 32 x 24 cm`，均超过常见廉航免费随身行李尺寸，基础款也超过多数 `55 x 40 x 20/23 cm` 机舱行李厚度限制。
- 托运：`MT100 50L` 尺寸低于国内常见托运行李上限 `40 x 60 x 100 cm` 或三边和 `203 cm`，托运通常没问题；廉航机票常不含或只含较低托运行李额，订票时提前购买 `20 kg` 左右托运行李，比机场补买更稳。

## 采购约束

- 新购装备总额优先控制在 `800-1500 CNY`。
- 能借就借，能用已有就不买。
- 鞋必须提前磨合。
- 出发前称重，背包不含水和食物控制在 `7-10 kg`。
