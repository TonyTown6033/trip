# Yunnan Backpacking Trip

云南两个月背包旅行项目。默认设定为 `12000 CNY` 总预算、约 `60 天`、单人背包客、低成本交通、青旅/低价客栈优先。

这个目录按“途中使用”设计：每天先看 `00-dashboard/`，到某一站时打开 `03-stages/<stage>/`。

## 目录结构

- `agent.md`：Agent 工具、MCP、数据源和执行边界约定。
- `00-dashboard/`：途中首页，放总览、预算余额、待办和紧急信息。
- `01-gear/`：装备清单、采购清单、重量检查。
- `02-route/`：总路线、路线地图、跨站备选方案。
- `03-stages/`：每一站的执行资料，是路上最常用的目录。
- `04-bookings/`：人工确认后的订单和预订信息。
- `food/`：餐馆、小吃、市场和日常吃饭候选，作为 Web 可展示的一级资料。
- `05-receipts/`：票据、发票、付款凭证。
- `06-research/`：通用工具、MCP、天气源和历史查价。
- `07-exports/`：导出的表格、PDF、地图、分享版本。
- `templates/`：复用模板。

## 预算红线

- 总预算：`12000 CNY`
- 装备新购：`800-1500 CNY`
- 旅行中可花预算：约 `10500-11200 CNY`
- 60 天日均：约 `175-187 CNY`
- 稳妥执行目标：日均 `150-170 CNY`

## 每天怎么用

1. 打开 `00-dashboard/overview.md` 看当前位置、下一站、预算余额和风险。
2. 打开当前站点目录，例如 `03-stages/02-dali/`。
3. 更新当前站点的 `plan.md`、`budget.csv`、`risk.md`。
4. 查询交通、住宿、天气时，把来源和时间写入对应文件。
5. 已下单的信息只放 `04-bookings/`，票据放 `05-receipts/`。

## 省钱原则

- 青旅床位、低价客栈、连住优惠优先。
- 火车、公交、班车、拼车优先。
- 每天只安排 `0-1` 个付费景点。
- 高价索道、旅拍、商业演出、高价一日团默认不安排。
- 包车只在公共交通不可用且安全必要时使用。
