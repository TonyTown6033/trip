# Plan

## 阶段定位

- 沙溪/剑川是大理之后的慢节奏古镇段，建议 3-4 晚，不以 `2026-05-29` 作为返程或硬截止。
- 主目标：住沙溪古镇内，降低县城往返成本；用 1-2 天完整慢逛沙溪，天气稳定时再决定石宝山。
- 剑川县城作为交通缓冲，不作为每天往返沙溪的默认住宿方案。
- 雨季原则：不安排夜间山路，不为石宝山临时包车硬去，不把远端山地和转场放在同一天。

## 居住计划

主住沙溪古镇，县城只作到达/离开过渡：

1. 沙溪路元文化中心客栈：`¥42`，沙溪古镇寺登街，近沙溪古镇，主选。
2. 大理沙溪颖居客栈：`¥75`，沙溪古镇寺登街，古镇内舒适型备选。
3. 大理剑川海缘客栈：`¥35`，剑川古城/县城最低价，适合晚到或早走过渡。
4. 剑川文献德玉宾馆(原文献德玉旅馆)：`¥65`，县城过渡备选。
5. 剑川e家酒店：`¥90`，县城应急备选，只有古镇低价房满房或需贴近车站时考虑。
6. 大理剑川邮电宾馆：`¥116`，超过常规预算，只作满房应急。

默认订沙溪古镇内 2 晚可取消房源，到店后再根据天气、石宝山交通和下一站丽江班车情况决定是否续住第 3-4 晚。

## 到离交通

- 到达：大理 -> 沙溪/剑川，白天班车或正规拼车。
- 推荐路径：大理汽车客运北站 -> 沙溪客运站；若无直达，改大理汽车客运北站 -> 剑川汽车客运站 -> 沙溪。
- 客运站信息：
  - 大理汽车客运北站：榆华路30号，电话 `0872-2292203`。
  - 剑川汽车客运站：剑川县人民医院斜对面，电话 `0872-4521311`。
  - 沙溪客运站：幸福巷与022乡道交叉口西南340米。
- 路线耗时参考：
  - 大理汽车客运北站 -> 沙溪客运站，高德驾车约 `165.2 km / 2 小时 03 分钟`。
  - 沙溪客运站 -> 丽江客运站，高德驾车约 `100.6 km / 1 小时 16 分钟`。
  - 剑川汽车客运站 -> 丽江客运站，高德驾车约 `66.3 km / 55 分钟`。
- 离开：沙溪/剑川 -> 丽江，白天出发；优先确认沙溪客运站是否直达丽江，无直达则回剑川换乘。
- 镇内：沙溪古镇步行优先；石宝山、宝相寺、海云居等只在天气、道路、返程交通都明确时安排。

## 条件分支/删减规则

- 必保留：D1 白天转场、D2 沙溪古镇完整慢逛、离开前确认丽江交通。
- 可删减：石宝山、宝相寺、海云居、野生猴群、幽情谷等山地/石窟点。天气、道路、票价或交通任一不稳就取消。
- 到达太晚：不夜间继续赶沙溪；改住剑川县城候选，次日白天进古镇。
- 沙溪古镇低价房满房：先用大理沙溪颖居客栈 `¥75`；仍不可用再住剑川县城，按总成本而不是单晚房价决策。
- 雨天：只做寺登街、四方街、玉津桥附近短距离活动，增加客栈公共区/咖啡馆休整；不去石宝山。
- 去丽江无直达：从沙溪回剑川汽车客运站换乘，预留半天；不要压到傍晚再找车。
- 餐饮预算紧张：以火烤饼、小吃、米线、简餐为主；超过 `¥60` 的火塘/菌菇/私房菜只保留 1 次或跳过。

## 下单/出发前核验

- 住宿用真实入住日期重查，不使用任何假定返程日；确认可取消、最晚到店时间、是否能寄存行李、洗衣晾晒、热水、潮湿和噪声评价。
- 大理汽车客运北站是否有直达沙溪班车；若无，确认大理 -> 剑川 -> 沙溪的衔接、票价、末班和行李规则。
- 沙溪客运站是否有直达丽江班次；若无，确认沙溪 -> 剑川 -> 丽江换乘方案。
- 石宝山、宝相寺、海云居门票、开放、交通、返程和天气全部需当天核验，以官方当日票价/公告为准。
- 拼车只选正规、白天、价格明确的方案；雨天不临时包车走山路。
- 沙溪夜间餐饮少，D1 到达前准备水、干粮或确认客栈附近简餐。
