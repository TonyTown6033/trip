# Transport

## 到达/离开

- 丽江 -> 泸沽湖：班车优先，白天走山路；高德 REST 驾车测距 `186.6 km`、约 `3小时6分`，实际班车通常需更久，受雨季塌方、落石、修路影响。
- 泸沽湖 -> 香格里拉：高德 REST 驾车测距约 `388.1 km`、约 `4小时41分`，公共交通大概率需经丽江/宁蒗中转；不建议安排当天赶到德钦。
- 泸沽湖 -> 丽江 -> 香格里拉：稳妥方案，先白天返回丽江，再换动车去香格里拉；如天气差，优先在丽江缓冲。

## 雨季/山路待确认

- 出发前一天和当天早晨复查丽江、宁蒗、泸沽湖沿线暴雨、地质灾害、道路管制。
- 遇连续降雨、客运站提示道路风险、车辆只剩夜间班次时，取消泸沽湖段。
- 不为了省钱坐黑车或夜间拼车；必须选择正规客运/可追踪拼车。
- 若从泸沽湖直转香格里拉信息不稳定，默认回丽江中转。
