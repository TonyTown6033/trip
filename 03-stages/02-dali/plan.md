# Plan

## 阶段定位

- 大理是第一段慢住核心站，预计 7-9 晚，基于昆明 `2026-05-25` 动车到达推算，先按 D1-D9 排可执行日程。
- 主目标：住在大理古城，低成本吃住，串联大理古城、才村/龙龛、喜洲和少量付费/天气窗口项目。
- 不把 `2026-05-29` 当返程节点；整趟旅行约 60 天，大理之后继续去沙溪/剑川。
- 节奏原则：到达日不赶景点；每 2-3 天放一个洗衣/休整/雨天室内或近距离日。

## 居住计划

主住大理古城，不住下关长期通勤；喜洲只作一日往返或短住体验，不作为本轮主住宿区。

1. 大理不觉晓青旅：`¥30`，古城东门村，步行到大理古城约 `982m / 19min`，主选。
2. 大理车马店·客栈：`¥36`，古城平等路/北侧低价客栈，靠近市场，适合低成本补给；需人工核准确地址。
3. 大理古城天怡客栈：`¥47`，一塔路/南门方向，位置和价格平衡。
4. 大理壹賜偶遇国际青旅：`¥52`，古城核心青旅，若公共区、卫生和洗衣更好可接受。
5. 大理暖暖精品青旅：`¥60`，位置可用但偏高，只在条件明显更稳时选。
6. 登巴伴山国际青年客栈：`¥47`，苍山方向，雨天和夜间回程不确定，排在古城低价候选后。
7. 大理银芳主题客栈(火车站店) `¥52`、屿她女生青旅（洱海公园店）`¥76`：只作大理站晚到/早走过渡，不作 7-9 晚主住。

下单思路：先订可取消的 2-3 晚试住；到店后确认卫生、噪声、洗衣、床位和公共区，再决定是否续住到离开沙溪前。

## 到离交通

- 到达：昆明 -> 大理动车优先，暂按 `2026-05-25` 查询，昆明站二等座多为 `¥145.0`，以 `12306` 当日票价为准。
- 推荐车次：
  - `D8730` 昆明 -> 大理，`10:09-12:11`，首选。
  - `D3932` 昆明 -> 大理，`11:44-13:52`，退房后更从容。
  - `G2836` 昆明 -> 大理，`13:30-15:38`，午后稳妥。
  - `D3956` 昆明 -> 大理，`16:50-18:48`，最晚可接受，需准备夜间进古城方案。
- 大理站 -> 大理古城：公交/公共交通优先；若晚到、下雨或行李重，把打车/网约车作为备选。
- 城内：古城内步行，去才村/龙龛/喜洲优先公交或正规公共交通；电动车需确认天气、路况、押金和保险。
- 离开：大理 -> 沙溪/剑川，白天从大理汽车客运北站出发，优先查是否直达沙溪；无直达则大理 -> 剑川 -> 沙溪。
- 路线参考：大理汽车客运北站 -> 沙溪客运站，高德驾车估算约 `165.2 km / 2 小时 03 分钟`，不等于班车耗时和票价。

## 条件分支/删减规则

- 必保留：D1 到达入住、D2 古城熟悉、D3 洱海边低强度、D4 喜洲、D9 白天去沙溪/剑川。
- 可删减：崇圣寺三塔、苍山/索道、双廊/小普陀/理想邦。预算或天气不合适时全部不硬排。
- 下雨/风大：取消洱海边长距离步行、环海东路、电动车和苍山索道；改古城、市场、洗衣、资料整理。
- 公交末班不清：喜洲和远端点提前返程，不安排夜间回古城。
- 住宿试住不满意：只保留 2-3 晚，换到同古城片区候选；不要迁到下关长期通勤。
- 餐饮预算超支：野生菌火锅/体验餐最多 1 次；日常回到 `¥15-30` 米线、饵丝、市场补给。
- 去沙溪班车不明：D8 下午或 D9 早上电话确认大理汽车客运北站 `0872-2292203`；不确定则推迟一天，不走夜路。

- 大理住宿用真实入住日期重查连续 7-9 晚实付价；确认可取消、押金、床位/单间类型、洗衣晾晒、行李寄存、热水、储物柜、近期差评。
- 昆明 -> 大理购票前复查实际出发日余票，优先昆明站出发，价格以 `12306` 当日为准。
- 大理站到古城的公交线路、末班时间和雨天打车预算，在出发前一天确认。
- 喜洲公交往返时刻、末班车、雨天替代方案需到大理后再核验。
- 崇圣寺三塔、苍山索道、严家大院、南诏风情岛等付费点全部以官方当日票价/开放/预约为准。
